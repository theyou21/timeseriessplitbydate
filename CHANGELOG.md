# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-02-17
### Added
- Expanded README for the ML community with motivation, use cases, comparison table, and FAQ/gotchas.

- Added MkDocs site scaffolding and GitHub Pages deployment workflow.
